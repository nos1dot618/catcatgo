Language agnostic library API search engine. Inspired by [Hoogle](https://hoogle.haskell.org/). Utilizes [Tree-sitter](https://tree-sitter.github.io/tree-sitter/) for parsing.

> [!Important]
> Work in Progress

## Getting Started

```shell
go run .
```
